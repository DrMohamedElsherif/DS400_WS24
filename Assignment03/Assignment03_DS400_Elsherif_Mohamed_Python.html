The categories should be based on the following criteria: low for quality ratings below 5, medium for quality ratings of 5 and 6, high for quality ratings of 7 and 8, and prestigious for quality ratings above 8.

# a function to categorize wine quality
def categorize_quality(quality):
    if quality < 5:
        return "low"
    elif quality in [5, 6]:
        return "medium"
    elif quality in [7, 8]:
        return "high"
    else:
        return "prestigious"

# create a new column 'quality_cat'

7. Explain why logistic a regression is not a suitable model for this dataset.

Logistic regression is not suitable for modeling this dataset because of:

-Multi-classes: Logistic regression is designed for binary classification. This dataset has Multiclasses (Quality 3-9). Handling multiclasses may require other complex variants like multinomial or ordinal logistic regression.

-Imbalanced Classes: Given the observed class imbalance in the dataset as was shown in the bar plot above, logistic regression may produce biased predictions favoring the dominant class.

-Multicollinearity: The strong intercorrelations between some features (e.g., residual sugar and density) observed in the correlogram violate logistic regression assumptions.
